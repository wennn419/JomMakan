# 🍽️ JomMakan

A web-based food recommendation system developed using **PHP, MySQL, HTML, CSS, JavaScript, and Docker**.

The system helps users discover suitable meals in **Mount Austin, Johor Bahru**, based on their preferences. Users can search, compare, save favourites, and receive random food recommendations through an interactive and user-friendly interface.

---

# 📖 Project Overview

JomMakan is designed to solve the common problem of deciding what to eat. It allows users to explore different food options using various filters and recommendation features while providing administrators with tools to manage food and restaurant information efficiently.

---

# Features

## User Features

- 🔍 Search food by keyword
- 💰 Filter by budget
- 🍜 Filter by cuisine
- ⭐ Filter by rating
- 🍽️ View food details
- ⚖️ Compare multiple food items
- ❤️ Save favourite foods
- 🕒 View recently viewed foods
- 🎲 Surprise Me random recommendation
- 🔗 Share food links
- 👤 User registration and login
- 👤 Profile management

## Admin Features

- 📊 Admin Dashboard
- ➕ Add food
- ✏️ Edit food
- ❌ Delete food
- ➕ Add restaurant
- ✏️ Edit restaurant
- ❌ Delete restaurant
